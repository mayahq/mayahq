# 🤖 Maya Command Reference

## **Reliable Command Patterns (90-98% Success Rate)**

Maya works best with specific, structured commands. Use these exact patterns for reliable results.

---

## 📋 **Task Management**

### **View & Analyze Tasks**
```
"what's on my task list?"
"what should I work on next?"
"what should I prioritize today?"
"which tasks are most important?"
```
*Maya will analyze your tasks and provide intelligent recommendations*

### **Create Tasks**
```
"create task: [description]"
"add task: [description]"

Examples:
• "create task: fix the API authentication bug"
• "add task: review pull request #42"
• "create task: call client about project timeline"
```

### **Complete Tasks**
```
"mark task complete: [task name or ID]"
"complete task: [task name or ID]"

Examples:
• "mark task complete: fix API bug"
• "complete task: 123"
```

---

## ⏰ **Reminders**

### **Create Reminders** (Strict Format Required)
```
"remind me in [X] minutes to [action]"
"remind me in [X] hours to [action]"
"set reminder for [X] minutes to [action]"
"set reminder for [X] hours to [action]"

Examples:
• "remind me in 20 minutes to take a break"
• "remind me in 2 hours to call mom"
• "set reminder for 30 minutes to check on dinner"
• "set reminder for 1 hour to submit the report"
```

### **Casual Prefixes Supported**
```
"ok remind me in 15 minutes to stretch"
"please set reminder for 45 minutes to leave for meeting"
"hey remind me in 3 hours to water plants"
```

### **View Reminders**
```
"what are my upcoming reminders?"
"show me all my reminders"
"what reminders do I have?"
```

## 📅 **Calendar Events**

### **Create Events**
```
"add calendar event: [title] [time] at [location]"
"create event: [title] [time]"
"schedule: [title] [time] at [location]"

Examples:
• "add calendar event: Coffee Meeting 2pm at Starbucks"
• "create event: Team Standup 9am tomorrow"
• "schedule: Client Call 3pm at Conference Room A"
```

### **Manage Events**
```
"can you delete that event?"
"can you edit that event?"
```
*Works on your most recently created event*

### **View Events**
```
"what's on my calendar today?"
"show me upcoming events"
"what's my schedule this week?"
```


## 💡 **Tips for Best Results**

### **✅ DO:**
- Use exact patterns shown above
- Be specific with times (numbers + "minutes"/"hours")
- Include "to" before actions in reminders
- Use colons after "create task" and "add calendar event"

### **❌ AVOID:**
- Vague language: "remind me later about something"
- Missing words: "remind me 20 minutes take break"
- Wrong format: "set reminder to take break in 20 minutes"
- Casual requests: "maybe set up a reminder sometime"

---

## 🎯 **Priority Commands (Most Important)**

1. **Task Analysis**: `"what should I work on next?"` - Maya's core strength
2. **Quick Reminders**: `"remind me in 20 minutes to [action]"`
3. **Calendar Events**: `"add calendar event: [title] [time] at [location]"`
4. **Task Creation**: `"create task: [description]"`

---

## 🔧 **Troubleshooting**

**If a command doesn't work:**
1. Check the exact format above
2. Make sure you include all required words ("to", "in", ":")
3. Use numbers for time (not "twenty", use "20")
4. Try the command again with exact syntax

**Still having issues?** Say `"/help"` or ask Maya to show you the command reference.
